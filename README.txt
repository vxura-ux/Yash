Neon Portfolio — quick install (mobile / Acode)

1) Create a new folder in Acode (example: neon_portfolio).
2) Add these files exactly: index.html, style.css, animations.css, script.js, README.txt
3) Open index.html in Acode and press "Open in Browser" or preview.
4) The 3D slider, tilt, and animations run with JS — no external images or videos required.
5) To add images later: create assets/images/ and update <img src="..."> in HTML.
6) If anything breaks, paste the content here and I will fix it for you.
